Leaver is a Shelter plugin that allows you to batch leave Discord servers with automatic rate limiting and error handling.

Features:

  • View all servers you're currently in
  • Select multiple servers to leave at once
  • Automatic rate limiting to avoid API limits
  • Queue management with pause/resume functionality
  • Error handling with retry attempts
  • Real-time progress tracking

How to use:

Open the Leaver section from the discord settings sidebar to view all your servers. Select the servers you want to leave, add them to the queue, and start the leaving process. The plugin will handle rate limiting automatically and show progress for each server.
