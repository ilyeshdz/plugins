## Browser

I built Browser because I wanted one place to browse and install plugins and themes. Existing solutions either had clunky UIs or only handled plugins, leaving themes as an afterthought.

Browser just works. You open it, you browse, you install. It handles both plugins and themes with a clean interface that makes everything feel effortless. For themes, I built a simple system since Shelter doesn't provide one out of the box. Theme discovery, install, and switch are all there.

It turned out exactly how I wanted: simple, reliable, and actually pleasant to use.
